# Pace - Página Web Romántica

Espacio digital íntimo para celebrar y documentar momentos en pareja. Sin servidores, sin registro; todo se guarda en el dispositivo (LocalStorage).

## Estructura del proyecto

```
├── index.html          # Estructura principal
├── css/
│   └── styles.css      # Estilos
├── js/
│   ├── main.js         # Inicialización y eventos globales
│   ├── counter.js      # Contador de tiempo
│   ├── messages.js     # Mensajes de amor
│   ├── gallery.js      # Galería de fotos
│   ├── dates.js        # Fechas especiales
│   ├── wishlist.js     # Wishlist compartida
│   ├── audio.js        # Reproductor de música
│   └── storage.js      # Utilidades LocalStorage
├── assets/
│   ├── images/         # Imágenes del proyecto
│   └── audio/          # Archivos de audio
├── PRD.md              # Requisitos del producto
└── README.md
```

## Cómo usar

1. Abre `index.html` en un navegador moderno (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+).
2. Configura la fecha de inicio de la relación en el código (según PRD).
3. Añade mensajes, fotos y fechas especiales; todo se guarda localmente.

Requisitos completos y criterios de aceptación: ver **PRD.md**.
